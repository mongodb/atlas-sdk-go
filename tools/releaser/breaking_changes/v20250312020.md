# Breaking Changes

## SDK changes

### incompatible changes

- Dynamic JSON fields (`any`, `[]any`, `map[string]any`): numbers now decode as `json.Number` instead of `float64`; callers that type-assert to `float64` must migrate (see `docs/mig_1_json_precision.md`)
- DeprecatedStreamProcessorMetricThreshold.MetricName: changed from string to \*string
- DeprecatedStreamProcessorMetricThreshold: removed
- NewDeprecatedStreamProcessorMetricThreshold: removed
- NewDeprecatedStreamProcessorMetricThresholdWithDefaults: removed
